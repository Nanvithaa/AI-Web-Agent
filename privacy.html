This Chrome extension does not collect, store, or share any personal user data. All processing is done on-the-fly to summarize and answer questions based on the current webpage. The extension requires access to the active tab to extract visible text for generating responses. This data is only used temporarily to generate the response and is never saved or transmitted elsewhere. We do not use analytics, ads, or third-party tracking.
